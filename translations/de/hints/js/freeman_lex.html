Fangen Sie eine Idee 💡! Sie müssen nur ein paar Bedingungen aufstellen, um die folgende Regel auszudrücken. Das Jahr ist ein Schaltjahr, wenn es durch 4 teilbar ist und nicht durch 100 oder 400 teilbar ist.

Ok, womit soll ich anfangen?

Lesen Sie über den modulo (%) Operator und verwenden Sie ihn. Lassen Sie uns die erste Bedingung schreiben: Wenn das angegebene Jahr NICHT durch 4 teilbar ist (der Rest aus der Modulo-Division ist ungleich 0), kann Ihre Funktion sofort false zurückgeben, da es sich nicht um ein Schaltjahr handelt. 

if (year % 4!== 0) return false;
Was ist als nächstes zu tun?

Wenn ein bestimmtes Jahr die erste Bedingung erfüllt, prüfen Sie auf die gleiche Weise, ob es durch 100 teilbar ist. Wenn nicht, brauchen Sie das und Ihre Funktion kann true zurückgeben. 

if (year % 100!== 0) return true;
Ich brauche noch einen Hinweis.

Wenn es durch 100 teilbar ist, besteht die letzte Chance, ein Schaltjahr zu erhalten, wenn es durch 400 teilbar ist. Ihre Funktion muss das Ergebnis dieses Vergleichs zurückgeben. 

return year % 400 == 0;
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

function isLeapYear(year: number): boolean {

    if (year % 4!== 0) return false;
    if (year % 100!== 0) return true;
    return year % 400 == 0;
}

Klar, schnappen Sie es sich!

Verwenden Sie den verschachtelten ternären Operator und verbinden Sie alle Bedingungen in einer Zeile. 

function isLeapYear(year: number): boolean {

    return (year % 4 === 0)? ((year % 100!== 0)? true: (year % 400 === 0)): false;
}
