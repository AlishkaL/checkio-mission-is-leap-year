抓住一个点子💡！你只需要写几个条件来表达下面的规则。当年份能被 4 整除而不能被 100 整除或被 400 整除时，该年份为闰年。

好吧，我应该从什么开始呢？

阅读modulo (%) 运算符并使用它。让我们写下第一个条件：如果给定的年份不能被 4 整除（模除的余数不等于 0），函数可能会立即返回False ，因为这不是闰年。 

if year % 4!= 0: return False
接下来该怎么办？

如果给定的年份通过了第一个条件，用同样的方法检查它是否能被 100 整除。如果不能，这就是您需要的，您的函数可能返回True 。 

if year % 100!= 0: return True
我还需要提示。

如果它能被 100 整除，那么得到闰年的最后机会就是它能被 400 整除。您的函数必须返回这种比较的结果。 

return year % 400 == 0
那么，我的最终解决方案应该是怎样的呢？

您只需将之前所有的提示连接起来，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def is_leap_year(year: int) -> bool:

    if year % 4!= 0: return False
    if year % 100!= 0: return True
    return year % 400 == 0
