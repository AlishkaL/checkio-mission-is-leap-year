Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Wystarczy napisać kilka warunków, aby wyrazić następującą regułę. Rok jest przestępny, gdy jest podzielny przez 4 i nie jest podzielny przez 100 lub podzielny przez 400.

Ok, od czego powinienem zacząć?

Proszę poczytać o operatorze modulo (%) i go użyć. Napiszmy pierwszy warunek: jeśli podany rok NIE jest podzielny przez 4 (reszta z dzielenia modulo nie jest równa 0), funkcja może natychmiast zwrócić false, ponieważ nie jest to rok przestępny. 

if (year % 4!== 0) return false;
Co robić dalej?

Jeśli dany rok spełnia pierwszy warunek, proszę sprawdzić w ten sam sposób, czy jest podzielny przez 100. Jeśli nie, to właśnie tego potrzebujemy, a funkcja może zwrócić true. 

if (year % 100!== 0) return true;
Nadal potrzebuję podpowiedzi.

Jeśli jest podzielny przez 100, ostatnią szansą na uzyskanie roku przestępnego jest podzielność przez 400. Pana funkcja musi zwrócić wynik tego porównania. 

return year % 400 == 0;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function isLeapYear(year: number): boolean {

    if (year % 4!== 0) return false;
    if (year % 100!== 0) return true;
    return year % 400 == 0;
}

Jasne, proszę pobrać!

Proszę użyć zagnieżdżonego operatora trójargumentowego i połączyć wszystkie warunki w jednym wierszu. 

function isLeapYear(year: number): boolean {

    return (year % 4 === 0)? ((year % 100!== 0)? true: (year % 400 === 0)): false;
}
