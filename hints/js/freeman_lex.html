Ok, what should I start from?

Read about modulo (%) operator and use it. Let's write the first condition: if the given year is NOT divisible by 4 (the remainder from modulo division is not equal 0) your function may immediately return false, because it is not a leap year. 

if (year % 4 !== 0) return false;
What to do next?

If a given year passes the first condition, check in the same way if it's divisible by 100. If not, that's what you need and your function may return true. 

if (year % 100 !== 0) return true;
I still need a hint.

If it's divisible by 100, the last chance to get a leap year is if it's divisible by 400. Your function must return the result if this comparison. 

return year % 400 == 0;
So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function isLeapYear(year: number): boolean {

    if (year % 4 !== 0) return false;
    if (year % 100 !== 0) return true;
    return year % 400 == 0;
}

Use nested ternary operator and join all conditions in one line. 

function isLeapYear(year: number): boolean {

    return (year % 4 === 0) ? ((year % 100 !== 0) ? true : (year % 400 === 0)) : false;
